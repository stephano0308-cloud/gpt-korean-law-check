# Action Reflection Note

이 문서는 `gpt-korean-law-check`의 기능 변화를 Custom GPT에 반영할 때 참고하는 메모입니다.

## 현재 반영 대상
- 관련 법령 비교표 출력
- 연혁법령 세부 내역 출력
- 유사 조세심판원 선결정례 및 대법원 판례 비교분석
- 맞춤법·오탈자 검토

## Custom GPT에 반영할 항목
1. Instructions에는 `CUSTOM_GPT_INSTRUCTIONS.md`의 내용을 사용합니다.
2. Action에는 배포된 서버의 `openapi.json` 또는 `openapi.yaml`을 사용합니다.
3. 배포 서버가 여러 엔트리포인트로 분화되어 있는 경우, 실제 운영용 엔트리포인트가 제공하는 기능 범위를 먼저 확인합니다.
4. Railway가 GitHub `main` 자동배포 구조라면, 필요한 기능 브랜치의 변경사항을 `main`에 병합해야 실제 Action 결과에도 반영됩니다.

## 중요한 주의사항
- GPT 지시문만 바뀌어도 출력 형식은 개선되지만, 연혁 세부 내역/선례 비교/맞춤법 검토는 서버 응답 구조가 함께 반영되어야 완전하게 동작합니다.
- 따라서 기능별 브랜치가 `main`에 병합되었는지, 실제 배포 서버가 해당 엔트리포인트를 사용 중인지 확인해야 합니다.
