# Output Style Guide

## 1. 관련 법령 비교표
- 좌측: 사용자 파일상 내용 정리
- 우측: GPT 검토 결과 정리
- 우측에서 사용자 입력과 다른 부분은 반드시 `**볼드**` 처리한다.
- 파일에 없는 법령은 좌측에 `기재 없음`으로 적고 우측의 추가 법령을 `**볼드**` 처리한다.

## 2. 관련 법령 정리본
- 법령명은 가능한 경우 연혁 표시와 함께 적는다.
- 예: `소득세법(2005.12.31. 법률 제7837호로 개정되기 전의 것)`
- 조문은 `제○조(조문명)` 형식으로 정리한다.

## 3. 부칙
- 부칙이 문제되는 경우 `시행일`, `적용례`, `경과조치`를 나누어 적는다.
- 확인되지 않으면 `추가 확인 필요`라고 적는다.

## 4. 실무형 문구
- 추상적 설명보다 조사서에 바로 붙여넣을 수 있는 문장으로 정리한다.
- 불확실하면 단정하지 말고 `추가 확인 필요`라고 적는다.
